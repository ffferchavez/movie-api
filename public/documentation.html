myFlix API Documentation

Movies Endpoints

Request URL Response
Get all movies GET /movies List of all movies
Get a single movie by title GET /movies/:title Movie details
Get a genre by name GET /genres/:genreName Genre details
Get a director by name GET /directors/:directorName Director details

Users Endpoints

Request URL Response
Register a new user POST /users New user details
Update user information PUT /users/:userId Updated user details
Add a movie to user's favorites
