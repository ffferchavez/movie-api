Movie API

Welcome to the Movie API documentation!

This API allows you to access movie data and perform various operations related to movies.

Here are some of the main features of the API:

Operation Request Method URL Query Parameters Request Body Response
Retrieve a list of movies GET /movies N/A N/A List of movies
Search for movies GET /movies
  • title (string)
  • genre (string)
  • release_year (number)
 N/A List of movies matching the search criteria
Get detailed information about a specific movie GET /movies/{movie_id} N/A N/A Detailed information about the movie
Add new movies to the database POST /movies N/A 
{
    "title": "Movie Title",
    "genre": "Action",
    "release_year": 2022
}
Created movie object
Update movie information PUT /movies/{movie_id} N/A 
{
    "title": "Updated Movie Title",
    "genre": "Adventure",
    "release_year": 2023
}
Updated movie object
Delete movies from the database DELETE /movies/{movie_id} N/A N/A Success message
